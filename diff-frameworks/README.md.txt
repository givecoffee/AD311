

Gain hands-on experience with various React frameworks by setting up and developing basic applications. This assignment will help you understand the unique features and development workflows of Next.js, Remix, Gatsby, and Expo.
Setup Instructions:

Before starting, ensure you have the following installed on your computer:

    Node.js (latest stable version)
    A code editor of your choice (e.g., Visual Studio Code)
    Git (for version control)

Tasks:

    Next.js Application:
        Create a new Next.js project using npx create-next-app@latest your-next-app.
        Optional: Develop a simple landing page that includes a header, a few sections with placeholder text, and a footer.
        Optional: Implement Static Site Generation (SSG) for the landing page. (npm run build)
        Optional: Add a dynamic route to display user profiles fetched from a public API (e.g., https://jsonplaceholder.typicode.com/users).

    Remix Application: (Optional)
        Initialize a new Remix project with npx create-remix your-remix-app.
        Optional: Create a blog application with a homepage listing blog posts and individual pages for each post.
        Optional: Use Markdown files as the source for blog posts, and parse them at build time to generate static pages.
        Optional: Implement nested routing for categories or tags within your blog.

    Gatsby Application:
        Start a new Gatsby project with npx create-gatsby your-gatsby-app.
        Optional: Build a portfolio site showcasing projects or artworks. Use static assets and Gatsby's GraphQL data layer to manage your project data. 
        Optional: Style your site using Gatsby's CSS-in-JS support or a CSS framework of your choice. 
        Optional: Integrate a CMS (e.g., Contentful, Sanity) to manage your portfolio content dynamically.

    Expo Application:
        Create a new Expo project using npx create-expo-app your-expo-app.
        Optional: Develop a simple mobile app that features a tab navigator with at least two tabs: Home and About. 
        Optional: On the Home tab, display a list of items fetched from an external API. 
        Optional: Implement push notifications that alert users when new items are added to the list.

 
Assignment Submission:

Once you have completed the assignment, please prepare the following for submission:

    GitHub Repository:
        Push your program to a new GitHub repository.
        Ensure that your repository is public so that it can be accessed and reviewed.
        Your repository should include all source code files and a README.md file that briefly describes the project and how to run it.

    Demonstration:
        Provide a video demonstrating your program in action. The demonstration should include:
            Show the program's functionality.

    Submission:
        Submit the link to your GitHub repository.
