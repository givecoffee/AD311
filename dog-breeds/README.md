# Dog Breeds API

A vanilla JavaScript app for exploring dog breeds, facts, and groups using the [Dog API](https://dogapi.dog/api/v2). No frameworks, no build tools, no dependencies. Just open the file and go.

---

## Features

| # | What it does |
|---|---|
| 1 | Loads all dog breeds on startup (paginated, cached) |
| 2 | Click any breed card to see a detail panel (life span, weight, hypoallergenic status, description) |
| 3 | Random dog facts on demand |
| 4 | Browse breeds organized by group — collapsible cards, client-side filtered |
| 5 | In-browser test suite with normal and edge case coverage |

---

## How to Run

Open index.html in web browser.
